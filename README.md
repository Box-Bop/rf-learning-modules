# rf-learning-modules


## How to run this app

```
npm i

docker-compose up
```

The app will start running on port 3002
<br></br>
### **Available endpoints:**

* `/getModulesByCategoryName/:category`  
	search for modules by quering the category name
* `/getPopularModules`  
	returns the top 10 modules used in the last month
